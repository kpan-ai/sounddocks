Sounddocks

A clean soundboard for PC. Play sounds in Discord voice calls with hotkeys, import from MyInstants, and never touch Voicemeeter.

⬇ Download for Windows ◈ View on GitHub

Requires VB-Cable (free) · Windows 10/11

FEATURES

Everything you need,
nothing you don't

⌨
Global hotkeys
Bind any key combo to a sound. Hotkeys fire even when the app is minimized or in the background.
⬇
MyInstants import
Paste any myinstants.com URL and the sound downloads and appears in your board instantly.
🎚
Dual audio output
Route sounds to Discord and your headphones separately. You hear everything, your friends hear everything.
💾
Auto-saves everything
Your sounds, hotkeys, and settings are saved automatically. Pick up right where you left off.
🎵
Any audio format
Import MP3, WAV, OGG, FLAC and more. Just drag your files in or use the file picker.
◈
Clean dark UI
A minimal, distraction-free interface. Rename sounds, change emojis, search your library.
SETUP

Up and running
in 5 minutes

1
Install VB-Cable
Download and install VB-Cable (free). Restart your PC after installing.
2
Set CABLE Output as default mic
Right-click the speaker icon → Sound settings → Input → set default to CABLE Output (VB-Audio Virtual Cable).
3
Route your real mic into the cable
Sound settings → More sound settings → Recording tab → right-click your real mic → Properties → Listen tab → check Listen to this device → set playback to CABLE Input.
4
Set Discord input to CABLE Output
Discord → Settings → Voice & Video → Input Device → CABLE Output (VB-Audio Virtual Cable).
5
Install Sounddocks and you're done
Run the installer. The app auto-detects CABLE Input and sets itself up. Add sounds, set hotkeys, play.
